# Pascale Content Engine 🚀

Sistema automatizado para mineração, edição e postagem de conteúdo para as marcas Pascale Tours e Pascale Regularização.

## ✨ Funcionalidades
- Download automático de Reels/TikTok.
- Crop inteligente para 9:16 (Vertical).
- Geração de legendas via Gemini AI.
- Branding dinâmico.

## 🛠️ Como usar
1. Instale o Python 3.10+
2. Instale o FFmpeg no seu sistema.
3. Rode: `pip install -r requirements.txt`
4. Inicie: `streamlit run app.py`
